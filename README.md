# Análise de Dados Públicos para Identificar Padrões de Evasão Escolar

## Projeto

Projeto desenvolvido para a disciplina Atividade Extensionista II do curso CST em Ciência de Dados da UNINTER.

## Objetivo

Analisar dados públicos relacionados à evasão escolar utilizando conceitos de Ciência de Dados para compreender padrões e apoiar decisões na área educacional.

## ODS Relacionados

- ODS 4 – Educação de Qualidade
- ODS 10 – Redução das Desigualdades

## Tecnologias Utilizadas

- Python
- Google Colab
- Pandas
- Matplotlib
- GitHub

## Fonte dos Dados

Instituto Nacional de Estudos e Pesquisas Educacionais Anísio Teixeira (INEP)

## Dados Utilizados

| Etapa de Ensino | Taxa de Evasão (%) |
|-----------------|--------------------|
| Ensino Médio | 11,1 |
| Ensino Médio (período posterior) | 9,1 |
| Fundamental - Anos Finais | 4,3 |
| Fundamental - Anos Iniciais | 1,5 |

## Resultados

Os dados analisados demonstraram que a evasão escolar apresenta índices mais elevados no Ensino Médio e nos anos finais do Ensino Fundamental.

Também foi observada redução da evasão escolar no Ensino Médio de 11,1% para 9,1%.

## Aplicação na Comunidade

Os resultados foram organizados em gráficos e apresentados de forma simplificada para demonstrar como a Ciência de Dados pode auxiliar na compreensão de problemas educacionais.
